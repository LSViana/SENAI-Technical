PRE-REQUISITES:
• Node JS
• Java 8 Development Kit
• Eclipse Java EE Oxygen

1 - You must run the commands "npm install" and "npm run start_std" inside the folder "Views";

2 - Inside Eclipse, start the Apache Tomcat server with the project SStorage loaded at that, make sure the URL path to this project is "/sstorage";

3 - Both servers, the Node JS, to supply HTML and assets, and the WebService project, to supply the API access, must be running at the same machine for evaluation purposes.
